# Print Shop Mobile App - Landing Page

A React Native cross-platform mobile app.

## Color Palette

The app uses a consistent color scheme stored in `colors.js`:

1. **#5A7D7C** - Primary color (balance card, active navigation)
2. **#DADFF7** - Background
3. **#232C33** - Primary text
4. **#A0C1D1** - Credit Wallet card
5. **#B5B2C2** - Print Request card
